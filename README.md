## 🚩진행방식

* 늦어도 목요일, 월요일 오전까지 문제를 뽑아서 폴더 생성한다.
* 한 사람당 한 문제씩 뽑아서 총 4문제를 풀어온다.
* 어떻게 풀었는지 자세하게 주석을 첨부한다.
* 랜덤으로 각자 1문제를 배당받고, 기술면접이라 생각하고 알고리즘 문제 풀이 방법을 설명한다. (PT면접 대비)
